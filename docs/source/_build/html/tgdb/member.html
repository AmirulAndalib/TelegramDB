Member

class telegramdb.Member(_: type, is_primary: bool = False)

Bases: object

Member of a DataPack.

Parameters

  • _ (type) – Datatype of the member.

  • is_primary (bool) – Whether the member is a primary key or not.

Example

from telegramdb import Member

# user_id of type int as a primary key
user_id = Member(int, is_primary=True)

# text of type str
text = Member(str)
